# **App Name**: EvalAI

## Core Features:

- Panel Login: Frontend-only password protection for the panel.
- Contestant Search: Search for contestants by roll number and display their details.
- Evaluation Input: Input field for marks and a textarea for evaluation feedback, updating Firestore on save.
- AI Question Suggestions: Use AI to dynamically suggest interview questions based on the contestant's preferred position. The AI tool will reason about what knowledge should be used for a given question, if any.
- Admin Login: Frontend-only password protection for the admin dashboard.
- Dashboard Visualizations: Display charts and graphs representing participant data, evaluation status, and score distributions.
- Contestant Listing: Table listing all registered contestants with search, filter, and sorting capabilities.

## Style Guidelines:

- Primary color: Deep Indigo (#66347F) to convey professionalism and intelligence.
- Background color: Very light gray (#F4F2F5), almost white, desaturated from the indigo to maintain a clean backdrop.
- Accent color: Light Lavender (#E6E6FA) - a lighter analogous color to subtly highlight key elements.
- Body: 'Inter' sans-serif font for a clean and modern reading experience.
- Headline: 'Space Grotesk' sans-serif font for headers to give a modern techy look, pairing well with Inter. 
- Use minimalist icons for dashboard elements and table actions.
- Admin-like dashboard layout with clear sections for charts, tables, and data display.
- Subtle transition effects for loading states and data updates to enhance UX.
